# PersoTWCSS

Well, wondering the hell that means? Perso = Personal, TWCSS = Tailwind CSS.

Personal website built with Tailwind CSS v4 and Vite.
This is my first coded website. I know it's not much, it's still work in progress. I'll add more to it as I learn more.


## Tech Stack

- [Tailwind CSS v4](https://tailwindcss.com/) — utility-first styling
- [Vite](https://vite.dev/) — dev server & production bundler

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```
